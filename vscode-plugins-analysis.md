# Visual Studio Code 插件用途分析

## 编程语言支持

### C/C++ 开发
- **ms-vscode.cpptools** - Microsoft C/C++ 扩展，提供智能感知、调试、代码导航
- **ms-vscode.cpptools-extension-pack** - C/C++ 扩展包，包含多个相关工具
- **ms-vscode.cpptools-themes** - C/C++ 主题包
- **franneck94.c-cpp-runner** - C/C++ 代码运行器
- **vadimcn.vscode-lldb** - LLDB 调试器支持
- **qiumingge.cpp-check-lint** - C++ 代码检查和格式化，集成了 clang-tidy、clang-format 等工具，提供静态代码分析和代码风格检查
- **kaysonwu.cpptask** - C++ 任务管理

### Python 开发
- **ms-python.python** - Python 语言支持
- **ms-python.vscode-pylance** - Python 语言服务器，提供更好的智能感知
- **ms-python.debugpy** - Python 调试器
- **ms-python.vscode-python-envs** - Python 环境管理

### Java 开发
- **vscjava.vscode-java-pack** - Java 扩展包
- **vscjava.vscode-java-debug** - Java 调试器
- **vscjava.vscode-java-dependency** - Java 依赖管理
- **vscjava.vscode-gradle** - Gradle 构建工具支持

### CMake 支持
- **ms-vscode.cmake-tools** - CMake 工具支持
- **twxs.cmake** - CMake 语言支持

## 总结

您的插件配置显示您主要专注于：
1. **C/C++ 和 Python 开发** - 有完整的开发环境支持
2. **AI 编程助手** - 使用了多个 AI 代码补全工具
3. **竞赛编程** - 有专门的竞赛编程插件
4. **机器人开发** - 配置了 ROS 和 URDF 相关工具
5. **远程开发** - 支持 SSH 和 WSL 开发环境

这是一个非常全面的开发环境配置，适合算法竞赛、系统开发和机器人编程等多种场景。
